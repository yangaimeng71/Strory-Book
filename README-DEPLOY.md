# 🎨 AI绘本 - 部署说明

## 🎯 快速获得在线链接的方法

### 方法1: Netlify（推荐，最简单）
1. 访问：https://netlify.com
2. 直接拖拽包含 `index.html` 的文件夹到页面
3. 立即获得在线链接，如：`https://amazing-ai-story.netlify.app`

### 方法2: GitHub Pages
1. 访问：https://github.com
2. 创建新仓库（public）
3. 上传 `index.html` 文件
4. 在 Settings → Pages 启用
5. 获得链接：`https://你的用户名.github.io/仓库名`

### 方法3: Vercel
1. 访问：https://vercel.com
2. 导入项目或上传文件
3. 自动部署获得链接
